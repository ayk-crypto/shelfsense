# ShelfSense

Smart inventory and expiry tracking SaaS for small businesses.

## Workspaces

- `apps/web` - React + Vite + TypeScript web app
- `apps/api` - Node.js + Express + TypeScript API
- `packages/shared` - Shared TypeScript types

## Scripts

```bash
npm run dev:web
npm run dev:api
npm run dev
npm run build
```
